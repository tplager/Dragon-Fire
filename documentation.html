Notes

All art assets were either taken from opengameart.org or, in the case of the buttons and other UI assets, made by myself with the aid of the Charmonman font from google fonts.

As you can see, the dragon is animated from a spritesheet.

The player also has the option to choose different dragons on the instructions page by clicking on them. All dragons are animated. The player's choice is saved in local storage and is by default, the one-headed red dragon.

The background is parallax and is animated.

The object of the game is to destroy as many knights as possible in the 60 seconds allotted. The remaining time is shown in the top right corner

The score is shown in the top left corner and the remaining health is shown below it. The total score at the end is the sum of these values.

The player can hit p to pause the game.

Once the game is over, the player has the option to restart and go back to the instructions screen, allowing them to choose another dragon if they wish and then start over.

I used the resizing tutorial shown above to create a function that resizes the game screen if the window is resized.

The sounds listed above are all links to the source for that particular sound. I used the howler docs to edit how the sounds were played in the js file.
